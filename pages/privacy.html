Privacy Policy

Last Updated: November 19, 2025

Introduction

Readable Spokable PDF ("we", "our", or "the application") is committed to protecting your privacy. This Privacy Policy explains how we handle your information when you use our browser-based PDF transformation service.

üîí
Privacy First: This application is designed with privacy as a core principle. All processing happens locally in your browser, and we do not operate any servers that collect or store your data.

1. Information We Do NOT Collect

Because this is a purely client-side application that runs entirely in your browser, we do NOT collect, store, or have access to:

  • Your PDF files or their contents
  • Your personal information
  • Your browsing history
  • Your location data
  • Usage analytics or statistics
  • Cookies (except for essential LocalStorage data)
  • IP addresses or device identifiers

2. Data Storage in Your Browser

The application stores data locally in YOUR browser using standard web technologies:

LocalStorage

  • API Keys: Your Google AI Studio API keys (encrypted by your browser)
  • Settings: Your configuration preferences (batch size, prompts, PDF settings, etc.)
  • Theme Preference: Whether you prefer light or dark mode

IndexedDB

  • PDF Files: Temporarily stored for processing
  • Batch Data: Intermediate transformation results
  • Processing Logs: Application logs for diagnostics
  • Session Data: Processing session state for recovery
üíæ
All of this data is stored ONLY on your device and is accessible only to this application in your browser. You can delete all stored data at any time using the "Erase All Data" button in Settings.

3. Third-Party Services

To function, the application communicates with the following third-party services:

Google AI Studio / Gemini API

  • Purpose: AI-powered text transformation
  • Data Sent: Text content from your PDF and images (if multimodal processing is enabled)
  • Authentication: Your API key is sent in request headers
  • Privacy Policy: Google Privacy Policy

Content Delivery Networks (CDNs)

The application loads the following libraries from CDNs:

  • PDF.js (Mozilla) - for PDF processing
  • jsPDF - for PDF generation

These are loaded once when you first visit the page and may be cached by your browser.

4. API Keys and Authentication

Your Google AI Studio API key is:

  • ‚úÖ Stored only in your browser's LocalStorage
  • ‚úÖ Never transmitted to any server operated by us (we don't have servers)
  • ‚úÖ Only sent to Google's Gemini API when making transformation requests
  • ‚úÖ Redacted in diagnostic exports (only first and last few characters shown)
  • ‚úÖ Clearable at any time via Settings

We recommend treating your API key as sensitive information and not sharing it with others.

5. Data Security

We take security seriously:

  • Client-Side Processing: All operations happen in your browser
  • HTTPS Communication: All API calls to Google use secure HTTPS
  • No Server Storage: We don't have servers, so your data can't be breached from our end
  • Browser Security: Data is protected by your browser's security model
  • Optional Backup Keys: Failover support without exposing primary key

6. Your Rights and Control

You have complete control over your data:

  • Access: All data is accessible via browser DevTools or the Diagnostics page
  • Export: Download your settings, logs, and data anytime
  • Delete: Clear individual items or erase all data with one click
  • Modify: Change settings without restriction
  • Portability: Export settings and import them elsewhere

7. Cookies and Tracking

This application does NOT use:

  • ‚ùå Tracking cookies
  • ‚ùå Analytics services (Google Analytics, etc.)
  • ‚ùå Advertising networks
  • ‚ùå Social media tracking pixels
  • ‚ùå Third-party data collection scripts

We use only essential browser storage (LocalStorage and IndexedDB) for application functionality.

8. Children's Privacy

This application is not specifically directed at children under 13. We do not knowingly collect any information from children. If you are a parent or guardian and believe your child has used this application, you can clear all data using your browser's settings or the "Erase All Data" option in the application.

9. Data Retention

Data is retained in your browser until you explicitly delete it:

  • Settings: Persist until you reset them or clear browser data
  • Files & Batches: Remain in IndexedDB until you delete them
  • Logs: Can be manually cleared or set to auto-delete after a period

Clearing your browser's cache and data will also remove all application data.

10. Changes to This Policy

We may update this Privacy Policy from time to time. Any changes will be posted on this page with an updated "Last Updated" date. We encourage you to review this policy periodically.

Since the application is open source and can be self-hosted, this policy applies to the official hosted version. If you're running a modified or self-hosted version, different privacy practices may apply.

11. Contact Us

If you have questions about this Privacy Policy, you can:

  • Open an issue on our GitHub repository
  • Review our FAQ page
