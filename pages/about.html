About Readable Spokable PDF

Our Mission

Readable Spokable PDF is a browser-based application designed to make technical documentation more accessible by transforming it into natural, spoken-friendly content optimized for text-to-speech (TTS) systems.

We believe that everyone should be able to consume technical content in a way that works best for them. Whether you're driving, exercising, or simply prefer listening over reading, our tool makes technical PDFs easy to listen to.

How It Works

  1. Upload - Select a PDF file with selectable text (no OCR required)
  2. Extract - We extract text and images from your PDF using PDF.js
  3. Transform - Google's Gemini AI converts technical content to natural speech
  4. Generate - A new PDF is created with the transformed content
  5. Listen - Use the output with any TTS app like Moon+ Reader Pro

Technology Stack

We use modern, proven technologies:

  • PDF.js - Mozilla's excellent PDF rendering library for text extraction
  • Google Gemini AI - State-of-the-art language models for content transformation
  • jsPDF - PDF generation library for creating output documents
  • IndexedDB - Browser database for storing intermediate data
  • LocalStorage - For persisting user settings and preferences
